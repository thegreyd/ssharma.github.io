Opacity is a property that applies to all the child elements of the parent element for which it’s defined . That means text, images and anything in between. So here we do not use opacity. While it is still useful in many other cases (not mentioned here).

Previously I added a layer on top of the image and tweaked the opacity property of that layer in an image editor. This can be easily done in CSS. We have rgba values, where the fourth parameter is the opacity which ranges from 0.0 ( completely invisible/transparent) to 1 (fully opaque),  letting you tweak the color’s effect by virtue of it’s transparency. 

  • First reset the body – padding,margin,border values to 0   
  • In the body specify  – background image url
  • Then wrap a div around the body content with some id, just after the opening body tag and in CSS, set its height and width to 100%.   
  • Now set its background to any color of your choice, with its rgb value . At first it will be fully opaque and will lie over the body background. 
  • Add an ‘a’ , to make it 'rgba’ and add another value for transparency (within range 0-1) inside brackets with a comma. This makes it partially visible.
  • Tweak the last 'a’ value to your liking. 

This method essentially is the same as previous one but much easier and it needs only CSS.

Here is the code for reference –

<html>

<head>

<title>background-test</title>

<style type=“text/css”>

body{

padding:0;

border:0;

margin:0;

background-image:url(pattern.png);

}

div#container{

height:100%;

width:100%;

background-color:rgba(255,213,130,0.5);

}

</style>

</head>

<body>

<div id=“container”>

<h1>Tweaking Background with RGBa values</h1>

</div>

</body>

</html>
