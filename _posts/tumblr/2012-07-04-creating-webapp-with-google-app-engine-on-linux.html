After completing the Unit 1 of CS253 Web Application Engineering on Udacity, you have to make your web application with Google App Engine. There is no GUI program in the App-engine SDK on Linux, so it was a wee bit harder to set it up. This is how i did it on Linux Mint 13.

Note: This should also work for Ubuntu and other Ubuntu-based distros.

  • Sign in to : https://appengine.google.com/ . Create a new app.
  • Give a valid Application Identifier. In my case : udacious-cs.appspot.com
  • Give a title and click Create Application.
  • Python 2.7 should already be installed. To check: open Terminal and type python and hit Enter. It should display the version no. If not, run Update Manager.
  • Download Google App Engine SDK for Python, platform - Linux, from here:  https://developers.google.com/appengine/downloads . Unzip it. In my case :  ~/Downloads/google_appengine
  • Create a new folder for app files.  In my case :  ~/Downloads/projectapp
  • Create 2 files, app.yaml and a .py in the folder - with content from here - https://developers.google.com/appengine/docs/python/gettingstartedpython27/helloworld 
  • In my case: app.yaml and helloudacity.py.
  • Edit helloudacity.py to say ‘Hello, Udacity!’ instead of 'Hello, webapp World!’. (line 6)
  • Edit app.yaml , and replace 'application : helloworld’, with 'application : [Application-Identifier in step2]’ (line 1). In my case :  'application: udacious-cs’.
  • Edit app.yaml, and replace 'script: helloworld.app’ with 'script: [.py file name].app’ (line 9). In my case : 'script: helloudacity.app’.
  • To run the server and app locally : Open Terminal and run command(similar to): Downloads/google_appengine/dev_appserver.py Downloads/projectapp/ . Open http://localhost:8080/ in your browser to see your app working. 
  • You can edit .py file and the app will be updated in the browser. To stop server press Ctr+C in Terminal.
  • To deploy your app online (to update the new app you already created in step 1) : Open Terminal and run command(similar to): Downloads/google_appengine/appcfg.py update Downloads/projectapp/ 
  • Enter your google email and password, when prompted and hit Enter. The app online should be updated.
  • You should now see a dashboard for your app on https://appengine.google.com with a graph and other info.
  • Your app is ready : http://www.udacious-cs.appspot.com .
