These are pretty great resources for patterns/textures for background for your website/webpage/blog. 

  • Subtle patterns
  • Texture King
  • QTextures
  • CG Textures
  • Pattern8 

Learn how to add these patterns.

HOW TO -:

  • download the png/gif from the website of your choice.
  • don’t freak out because the image is so tiny, remember we want a pattern.
  • insert the following code in your css file (preferably in body{}).

background-image:url(path\name.png);

background-repeat:repeat;

(there is also a shorthand for it.. but i’m not going to mention it.)

Now most pattern i prefer are easy on the eyes, not to dominative. You may have difficulty in finding textures in colors other than different shades of grey ;they are the most used. If you want to add a shade/vibe of your own choice of color, you can easily do so. (I really liked a color as my background, but i wanted texture as well. The desired texture was nothing but grey(hey, i like grey but sometimes it becomes overwhelming!) So I wanted that texture with my bg color!). 

I am sure there are numerous other methods* to do the following, but you can say i have not entirely explored them;i’m not gonna mention them. (But if i feel some method being easier or better than this, i might update the post or write a new one)

HOW TO-:

  • Open the image in an image editor. I did this in Paint.net .
  • Add a new Layer above it and fill it with your desired color.
  • In the layer properties of the new Layer, adjust the  Opacity, to make it transparent so that the color merges with the texture.
  • Keep tweaking untill you get the right combination.
  • Save the new image as png/gif. Proceed to flatten the image.
  • Use the new image in your code.

*There are new options in CSS3 to set the opacity property in the code itself. And the RGBa values let you provide color with a transparent property. So I am trying find other ways to do this in the code itself without using image editor.
